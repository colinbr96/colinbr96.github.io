Othello

The classic board game recreated with Python

In one of my Information & Computer Science courses at UCI, “Programming with Software Libraries”, we learned how to code graphical user interfaces in Python using a library called TK. We were assigned a project to develop the game Othello and implement it using a responsive interface. This program is the result of two weeks of my hard work and dedication.

Phi Delt Cares

Phi Delta Theta’s philanthropy for the UCI CARE office

Phi Delt Cares was a philanthropic event hosted by my fraternity, Phi Delta Theta, at UCI during February of 2016. Our goal was to raise awareness and funds for the UCI Campus Assault Resources and Education (CARE) Office. The philanthropy ended successfully as we raised $700 by the end of the week! My personal duty for this event was designing the logo and managing the event pages.
